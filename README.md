## Project Overview

This repository contains all materials for the Anna Farm landscape installation project, including:
- Design documentation and client needs specifications
- HTML proposal documents for client review
- Reference photography and annotated imagery
- AI-generated concept renders (daytime/nighttime)
- Prompts and technical pipeline artifacts

## Design Concept

A site-specific geological sculpture inspired by the Colorado landscape:
- **Flatirons-inspired groupings** of vertical Colorado red flagstone slabs
- **Canyon cliff walls** of Colorado red strip stone along creek bends
- **Dry creek bed** of river rock flowing through the composition
- **Boulder anchors** of native Colorado granite and river boulders

## Key Standards

### Non-Deterministic Language
All proposal and documentation language uses flexible, non-deterministic terms:
- ✅ "a series of Flatirons-inspired groupings"
- ✅ "groupings of upright stone forms"
- ❌ "a single flat iron formation"
- ❌ "exactly X number of..."

### Materials (Colorado-sourced)
- Colorado red flagstone (NOT Arizona sandstone)
- Colorado red strip stone
- Native Colorado granite
- River boulders and cobbles

### Orientation
- Flat iron slabs point **northwest** (more west than north)
- North axis = sidewalk direction
- Visual intent: slabs face toward mountain context
